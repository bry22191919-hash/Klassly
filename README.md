# Klassly - A Google Classroom Clone

Klassly is a web application that allows teachers to create classes, students to join using class codes, and manage class content. 

## Features
- Teacher: Create classes, manage students
- Student: Join classes using a unique code
- Dashboard displaying all classes for a user
- Class page showing enrolled students
- Login and Signup authentication
- File upload and download

## Technologies
- Frontend: React, React Router
- Backend: Node.js, Express
- Database: SQLite
- Authentication: bcrypt

## Installation

1. Clone the repo

2. Install backend dependencies

3. Install frontend dependencies

4. Run backend(node server.js)

5. Run frontend(npm run dev)
